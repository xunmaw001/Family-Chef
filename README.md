# Family-Chef
“家庭大厨”微信小程序，一站式解决家庭烹饪难题。这里有海量菜谱，涵盖八大菜系及特色家常，按食材、口味、难度筛选超便捷，步骤图文与视频教程直观易学。支持一键添加食材至购物清单，联动生鲜平台下单配送。你还能上传成果，与厨友交流经验、参与烹饪挑战。小程序利用智能算法推荐个性化食谱，助你轻松变身家庭大厨，享受烹饪乐趣，为家人呈上美味佳肴 。
